## Unaided understanding loop (5 sessions)

1. Open project — do not open Settings or Help.
2. Read CMO thesis headline on Home.
3. Expand **Why this thesis** — confirm ≥1 repo or user evidence ref.
4. If sealing strategy, review **Evidence before you seal**.
5. Attempt Week 1 — note any FAB blocker toasts.

### Pass criteria

| Check | Expected |
|-------|----------|
| Why panel | Opens without backstage; shows confidence pills |
| Measured claims | Each has ≥1 evidence row |
| Seal gate | Blocks when business_model or activation missing |
| No fabrication | No competitor names without source |
