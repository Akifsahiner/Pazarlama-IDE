# Case study: Waitlist K-factor win (anonymous)

## Context
B2B devtool, 0 paid users, 180 Twitter followers. Launched waitlist with status unlock (not discount).

## Tactic stack applied
1. `waitlist_single_cta` — landing = email only
2. `referral_waitlist_loop` — invite 3 → early API access badge
3. `k_factor_instrument` — weekly K in sheet
4. `micro_launch_wave_3` — niche Slack, founder LI, newsletter swap

## Outcome
- 14 days: 412 signups, K = 0.34 after wave 2
- Referral accounted for 38% of signups week 2

## Lesson
Status unlock beat 20% discount — developers wanted access tier, not coupon.
