# Post-launch onboarding — 3 emails

| Send | Segment | Subject | Single action | Success metric |
|------|---------|---------|---------------|----------------|
| D+1 | Activated signup | Get [core outcome] in 10 minutes | Complete setup step 1 (link) | Step 1 done |
| D+3 | Completed step 1 | Most teams use [feature] daily | Try one workflow (template link) | Feature used once |
| D+7 | Any active user | Quick favor: how's week 1? | 1-question survey or reply | Response rate |

**Non-activated signups**
- D+2: "Stuck on setup?" — link to docs + reply-to founder
- Do not duplicate D+1 if already activated

## Copy skeleton (D+1)
```
Subject: Step 1 of getting value from [Product]

You signed up during launch week — here's the one thing to do today:
[Action] → [link]

Takes ~10 min. Reply if you hit a wall.

— [Founder]
```

## Acceptance
- [ ] One CTA per email
- [ ] Unsubscribe footer
- [ ] UTM `utm_campaign=onboard_d1|onboard_d3|onboard_d7`
