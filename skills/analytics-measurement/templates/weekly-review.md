# Weekly Review — Fill-In Template

30 minutes every Friday (or Monday — pick one, stick to it). **Decisions, not dashboards.** If this doc doesn't change what you do next week, you're tracking wrong metrics.

---

## Review header

```
Week ending:       [YYYY-MM-DD]
Owner:             [founder]
Attendees:         [founder / cofounder / AE if applicable]
hasAnalytics:      [true / false — manual log used?]
North-star:        [from kpi-tree — name + target]
```

## Example (filled — pre-launch week)

```
Week ending: 2026-07-11
North-star: Qualified signups (ICP-matched) — target 5/week

North-star: 4 (🟡 — one below target)
Leading: Founder calls booked 3 (🟢), LP visitor→signup 2.1% (🟡)

Diagnosis: Flat because PH traffic dropped post-launch-day; outbound produced 2/4 signups.

Channel: Outbound — 2 signups, 0 spend, 12% reply rate → HOLD, add 10 A-tier leads
Channel: Paid — paused (N too small)

Decision 1: Run lead-research sprint for 10 net-new A-tier accounts — owner Alex — success: 10 scorecards by Friday
Decision 2: Change LP headline to outcome variant from landing-page skill — owner Alex — success: signup rate >2.5%
NOT doing: Meta prospecting — N <300 for any test
```

## Bad vs good reviews

| Bad review | Good review |
|------------|-------------|
| "Traffic was okay" | "Organic flat at 40 sessions; outbound drove 2/4 signups" |
| 10 decisions for next week | Max 3 decisions with owners |
| Skipped when busy | 30 min non-negotiable — smallest viable review |
| Only vanity metrics | North-star + channel + qualitative |
| No NOT-doing list | Explicit deprioritization |

---

## Pre-launch vs growth additions

**Pre-launch (<100 users):** Add 5-second test result or founder call insight every week.

**With email list:** Add send performance row — open/click/signup per broadcast.

**B2B SaaS:** Add pipeline row — demos booked, SQLs, trial→paid snapshot.

---

## Reject the review if:

- North-star not scored vs target
- Zero decisions documented
- Channel table empty
- Manual log had >2 gap days without acknowledgment
- Review took <15 min (likely skipped sections)
