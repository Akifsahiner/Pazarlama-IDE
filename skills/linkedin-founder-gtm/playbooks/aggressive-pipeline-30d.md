# Playbook: Aggressive 30-day Founder Pipeline

Comment-led growth + DM sequences. Random posting fails; a grid compounds.

## Preconditions
- [ ] Founder LinkedIn profile exists
- [ ] ICP job titles known
- [ ] ≥3 hours/week for comments + posts

## Aggression dial

| Level | When | Focus |
|-------|------|-------|
| conservative | New profile | Headline + 3 posts/week |
| standard | Some network | 14-day grid + daily comments |
| aggressive | Warm network | Comment-led DM sequence + 70/20/10 |

## Timeline
- **T-30**: Headline/banner/featured link
- **T-21…T-7**: Comment 10 ICP posts/day before pitching
- **T-14**: Start 14-day content grid
- **T-7**: Soft CTA to waitlist/demo
- **H0…D+30**: DM only after 2 interactions

## Tactic stack
1. **`linkedin_14_day_grid`** — Batch 14 hooks; 70/20/10 mix
2. **`linkedin_comment_led_dm`** — Value-first DM after engagement
3. Measure reply rate; kill spray templates

## Orchestration
- Primary: LinkedIn founder presence
- Support: 1 waitlist or demo CTA only

## Realistic outcomes
- New profile: profile views ↑; few meetings first 14d — honest
- Warm: 5–15 qualified convos / 30d typical if ICP tight

## Kill / pivot rules
- Reply rate <2% after 40 DMs → rewrite opener; pause volume
- Vanity likes without ICP comments → change hooks

## Ethics line
- No fake engagement pods; no scraped InMail spam
