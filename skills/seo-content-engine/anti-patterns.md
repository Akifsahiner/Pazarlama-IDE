# Anti-Patterns

- **Don't:** Publish 50 AI-generated "best X tools" pages in one week. **Why:** Thin content penalty; one manual review flag can suppress the whole site section.
- **Don't:** Target head terms like "project management software" as a 2-person dev tool. **Why:** Zero rank probability; wastes sprint that could win "[niche] alternative to Notion."
- **Don't:** Copy competitor comparison tables verbatim. **Why:** Duplicate content + legal risk; buyers notice stale pricing.
- **Don't:** Hide pricing on comparison pages when competitors show it. **Why:** Commercial-intent queries bounce; trust loss is permanent.
- **Don't:** Stuff keywords in H1 ("Best AI SEO AI Tool AI 2026"). **Why:** Reads as spam; dev ICP exits immediately.
- **Don't:** Claim "#1 on Google" without GSC proof. **Why:** Violates honest metrics contract; destroys product trust.
- **Don't:** Buy backlinks or traffic packages. **Why:** Short-term spikes, long-term manual action risk — not Claude/Cursor-grade advice.
- **Don't:** Noindex docs because "we don't want SEO there." **Why:** Docs are the highest-trust organic surface for dev tools; gate only account/settings pages.
- **Don't:** Launch blog without sitemap + canonical tags. **Why:** "Discovered — not indexed" pile grows invisibly for months.
- **Don't:** Use the same meta description on all programmatic pages. **Why:** SERP cannibalization; Google picks none.
- **Don't:** Add comparison pages for competitors you've never used. **Why:** Factual errors get called out on HN/Reddit and become anti-marketing.
- **Don't:** Ignore Search Console coverage errors after deploy. **Why:** One broken redirect pattern can drop an entire cluster.
- **Don't:** Put signup modal on docs mid-article. **Why:** Docs readers are evaluating; interruption kills activation from organic.
- **Don't:** Merge blog and changelog without structure. **Why:** Informational and commercial intents compete; rankings dilute.
- **Don't:** Report SEO success with vanity impressions only. **Why:** 100k impressions at 0.1% CTR = failure; tie to signups or activated trials.
