# Case study: Dev tool comparison cluster → 2.4k organic/mo (anonymous)

## Context
B2B API monitoring tool. Stage: early growth. Assets: `/docs`, `/blog`, 6-person team, 2 named competitors (Datadog-style incumbent + newer OSS rival). Domain age 14 months. No paid SEO. Goal: capture "[incumbent] alternative" commercial intent.

## Tactic stack applied
1. `seo_gsc_baseline_audit` — 412 indexed URLs but only 89 with impressions
2. `seo_intent_cluster_map` — Tier A: 2 competitors × (alternative + vs)
3. `seo_competitor_serp_teardown` — SERP leaders used long tables + FAQ schema
4. `seo_alternatives_page_template` — honest table including OSS rival as best for self-host
5. `seo_vs_page_template` ×2 — pricing verified from public pages
6. `seo_how_to_hub_page` — "How to reduce alert noise without [incumbent]"
7. `seo_internal_link_graph` — docs sidebar → comparison pages
8. `seo_content_refresh_d30` — rewrote title on vs #1 page (CTR 1.1% → 3.4%)

## Outcome
- D+45: 4/5 cluster pages indexed
- D+90: **2,400 organic sessions/mo** (from 380 baseline)
- 62 trial signups attributed `google / organic` in month 3
- Top query: "[incumbent] alternative" — position 4–7

## Lesson
One tight cluster beat 12 months of random blog posts. Honest "choose incumbent if…" section reduced bounce and improved trust — sales cited comparison page in 8 deals.
