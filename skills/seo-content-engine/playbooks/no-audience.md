# Playbook: No Audience (SEO Runway)

No indexable blog/docs, no GSC, or domain too new for commercial pages. SEO starts with **infrastructure and one pillar** — not 20 programmatic URLs.

- **Week 1: Ship `/docs` or `/blog` route** in repo with sitemap entry + canonical. One pillar article: "How to [ICP job] without [pain]."
- **Week 2: Google Search Console** verify property; submit sitemap; fix any "Excluded" errors.
- **Week 3: Competitor SERP read** — manually open top 3 results for one niche query; note format (video, docs, comparison).
- **Week 4: First alternatives draft** — internal only until 1 docs page indexed.

Realistic outcome: 30–60 days before first meaningful impressions; 90+ days before organic signups matter.

## Preconditions
- [ ] Marketing profile lists real competitors
- [ ] Repo can ship at least one MDX/markdown route
- [ ] Honest aggression dial — no "rank in 2 weeks" promises

## Aggression dial

| Level | When | Focus |
|-------|------|-------|
| conservative | Single LP only | Docs route + GSC only |
| standard | Blog exists, no traffic | One pillar + internal links |
| aggressive | Not yet — use aggressive playbook when ≥3 routes indexed |

## Timeline
- **T-30**: GSC + sitemap
- **T-21**: Pillar how-to
- **T-14**: Internal link from LP footer to docs
- **T-7**: Competitor teardown notes
- **D+30**: Re-evaluate for aggressive cluster

## Orchestration
- One indexable surface at a time
- No programmatic until first URL indexed

## Realistic outcomes
- 60d: 50–300 organic sessions/mo on new domain
- Skipping GSC → flying blind on indexation failures

## Kill / pivot rules
- No index after 21d → engineering fix before content
- Pillar gets 0 impressions at 90d → rewrite angle for ICP job

## Ethics line
- No fake traffic tools to "warm up" domain
- No plagiarized docs content
