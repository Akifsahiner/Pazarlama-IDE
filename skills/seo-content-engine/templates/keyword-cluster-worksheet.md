## Intent definitions
- **Informational:** learner mode — docs/how-to
- **Commercial:** evaluating tools — alternatives, vs, pricing
- **Transactional:** ready to buy — pricing, signup, demo

## One cluster rule
All Tier A keywords must share semantic cluster (same competitor set + ICP job).
