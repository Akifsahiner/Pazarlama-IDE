# X Founder GTM

Dev founders live on X. This skill coordinates threads, build logs, launch amplification, and reply discipline — never engagement pods or vote-begging quote chains.

## Process
1. Map 30 ICP accounts (`x_icp_account_map`).
2. Fix bio + pinned thread (`x_profile_bio_hook`).
3. Ship 7-tweet thread template (`x_thread_format_7`).
4. Build-in-public cadence 3×/week (`x_build_in_public_cadence`).
5. Launch day: quote-tweet ethics + reply SLA (`x_quote_tweet_launch_ethics`, `x_reply_sla_2h`).

## Output
- 30-day X calendar (threads + singles + launch window)
- Paste-ready thread drafts with UTM
- Kill rules when engagement is vanity-only

## Ethics
- No engagement pods, no "like for like" rings, no begging for RTs
- Quote-tweet launch = add value, not "go upvote my PH/HN"
