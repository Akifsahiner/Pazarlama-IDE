# Lead Scorecard — Fill-In Template

Score every candidate 1–5 on five dimensions. **A-tier ≥20/25** with all must-haves met. Document notes — gut feel is not a score.

---

## Candidate header

```
Name:           [Full name]
Title:          [Current role]
Company:        [Name + domain]
LinkedIn:       [URL or "not found — contact via company page"]
Tier:           [A / B / C — auto from score + must-haves]
Source:         [linkedin_search / ph_maker / manual_seed / list_derived / etc.]
Source URL:     [link to post, PH profile, or search result]
Why-now:        [≤1 sentence — trigger event + date if known]
Research date:  [YYYY-MM-DD]
```

---

## Must-have gate (pass/fail before scoring)

All must pass for A-tier consideration:

| Must-have | Pass? | Notes |
|-----------|-------|-------|
| Role matches ICP persona | ☐ | |
| Company stage in ICP band | ☐ | |
| Industry / vertical fit | ☐ | |
| Identifiable pain (matches JTBD) | ☐ | |
| Why-now trigger ≤90 days | ☐ | |

**Any fail → max tier B.** Two fails → C (exclude from default outreach).

---

## Fit dimensions (1 = poor, 5 = perfect)

| Dimension | Score (1–5) | Notes |
|-----------|-------------|-------|
| **Role fit** — title owns the pain metric | | |
| **Company fit** — stage, size, industry | | |
| **Stack fit** — uses integrations you support | | |
| **Pain intensity** — public signal of active pain | | |
| **Timing** — trigger freshness and budget window | | |
| **TOTAL** | **/25** | |

### Scoring anchors

**Role fit**
- 5: Exact ICP title (e.g. Head of Ops when ICP = Head of Ops)
- 3: Adjacent title (Ops Manager, COO at small co)
- 1: Wrong function entirely

**Timing**
- 5: Trigger <14 days (new job, funding, launch)
- 3: Trigger 15–60 days
- 1: No trigger / speculative

## Tier assignment

| Total | Must-haves | Tier | Action |
|-------|------------|------|--------|
| ≥20 | All pass | **A** | Priority outreach draft |
| 15–19 | All pass | **B** | Outreach if A-tier exhausted |
| <15 or 2+ must-have fail | — | **C** | Exclude — do not draft |

## Bad vs good scoring

| Bad | Good |
|-----|------|
| "Seems like a good fit" — no scores | Each dimension 1–5 with note |
| A-tier with no why-now | Downgrade to B or document trigger |
| Score inflated because "we need more A leads" | Honest scores — 15 B beats 5 fake A |
| Email invented as gmail guess | LinkedIn URL or "email: unknown" |

- Source URL missing (evidence requirement)
- Score without notes on any dimension ≤3 or ≥4
