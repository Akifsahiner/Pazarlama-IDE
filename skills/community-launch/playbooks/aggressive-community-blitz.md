# Playbook: Aggressive Community Blitz

72-hour ethical maximum: Show HN spike + spaced IH + one Reddit value post. **Not** same-day spam across platforms.

## Preconditions
- [ ] Live demo URL tested on mobile + desktop (signup or core flow works)
- [ ] Show HN title drafted: `Show HN: [Name] – [outcome]` ≤80 chars
- [ ] First comment story written (problem → build → ask feedback)
- [ ] Subreddit rules audited for target sub (`comm_subreddit_rules_audit`)
- [ ] Founder calendar: 4h blocked for HN reply window
- [ ] UTM: `utm_source=hacker_news` / `reddit` / `indie_hackers`

## Aggression dial

| Level | When | Focus |
|-------|------|-------|
| conservative | No demo / lurker account | IH build log only; postpone Show HN 21d |
| standard | Demo live, some IH presence | Show HN + IH milestone (48h apart) |
| aggressive | Demo + prior HN comments + build log | This playbook — HN + IH + Reddit in 72h spaced |

**Honest ceiling:** Without live demo or HN comment history, front page is unrealistic. Expect 100–400 sessions on a average Show HN.

## Tactic stack

1. **`comm_icp_community_map` (T-14)** — List HN/IH/subreddits where ICP actually posts. Metric: 3 communities scored fit 1–5.
2. **`comm_subreddit_rules_audit` (T-14)** — Read rules + last 10 posts; note self-promo policy. Metric: checklist signed.
3. **`comm_show_hn_title_draft` (T-7)** — Title + first comment approved. Metric: title ≤80 chars, demo link in comment.
4. **`comm_demo_url_live` (T-7)** — E2E test signup/core flow. Metric: 0 blockers on mobile + desktop.
5. **`comm_ih_build_log` (T-10)** — IH post: build story, soft CTA in profile. Metric: published before Show HN.
6. **`comm_show_hn_submit` (H0)** — Tue–Thu morning PT. Metric: submission timestamp logged.
7. **`comm_hn_comment_sla_60m` (H0–H+4)** — Reply ≤60 min; factual on criticism. Metric: ≥90% SLA.
8. **`comm_cross_post_spacing_48h` (T-3)** — No HN+Reddit+IH same calendar day. Metric: calendar shows ≥48h gaps.
9. **`comm_reddit_value_post` (H+48)** — Story post per sub rules; link only if allowed. Metric: 0 mod removals.
10. **`comm_devto_companion_post` (H+72)** — Technical deep-dive + canonical URL. Metric: published with UTM.
11. **`comm_teardown_metrics_d1` (D+1)** — Sessions, signups, points, lessons. Metric: teardown live.

## Orchestration

- **Primary:** Hacker News Show HN (founder present 4h)
- **Parallel (spaced):** IH milestone → Reddit value → Dev.to
- **Do not:** PH same day, Discord mass ping, "upvote my HN post" tweets
- **Support:** Founder X/LinkedIn *teardown* only — not vote asks

## Realistic outcomes

| Profile | HN points (4h) | Sessions (72h) | Signups | Notes |
|---------|----------------|----------------|---------|-------|
| Aggressive (demo + history) | 150–400+ | 2k–15k | 150–1.2k | Front page possible |
| Standard (demo, cold account) | 40–120 | 400–2k | 40–200 | Solid awareness |
| No demo / wrong ICP | <30 | <200 | <20 | Use `no-audience` first |

## Kill / pivot rules

- H+2: <20 points and 0 substantive comments → do **not** cross-post Reddit; fix title/demo, retry in 21d
- Any mod removal → stop all posts 30d on that platform
- Signup rate <2% after 500 sessions → fix LP before next community spike
- Founder misses reply SLA → pause; reschedule Show HN

## Ethics line

- **Never:** vote brigading, alt accounts, paid upvotes, coordinated "go comment on my post"
- **Always:** story-first, live demo, respect subreddit and HN guidelines
- Aggressive = intensity of *legitimate* presence — not manipulation
