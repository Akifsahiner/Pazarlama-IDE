# Playbook: With Warm Community

You have IH followers, a niche subreddit reputation, or 50+ people who will genuinely comment on launch day — not a paid list.

- **T-14: DM 10 community friends** with draft Show HN title + ask "what's confusing?"
- **T-10: IH milestone** with real numbers (users, MRR, or waitlist — honest only).
- **T-7: Lock Show HN first comment** — mention community beta testers by role, not by name spam.
- **H0: Show HN Tuesday–Thursday 9am PT** — founder replies ≤60 min.
- **H+24: Thank-you IH post** linking teardown draft.

Do **not** ask community to upvote. Ask: "What would you want to see in v2?"

## Preconditions
- [ ] ≥10 people who will leave honest comments (not vote rings)
- [ ] Demo URL live
- [ ] Show HN title + first comment drafted

## Aggression dial

| Level | When | Focus |
|-------|------|-------|
| conservative | Small circle (<20) | IH only; delay HN |
| standard | 20–100 warm | Show HN + IH 48h apart |
| aggressive | 100+ engaged builders | Add spaced Reddit value post |

## Timeline
- **T-14**: Community map + DM feedback round
- **T-10**: IH milestone
- **T-7**: Demo QA + title freeze
- **H0**: Show HN
- **H+48**: Optional Reddit (rules OK)
- **D+1**: Teardown

## Tactic stack
1. **`comm_ih_build_log`** — Milestone with metrics
2. **`comm_show_hn_submit`** — Timed Show HN
3. **`comm_hn_comment_sla_60m`** — Reply SLA
4. **`comm_cross_post_spacing_48h`** — IH then HN then Reddit
5. **`comm_teardown_metrics_d1`** — Numbers public

## Orchestration
- IH warms audience; HN is spike; Reddit only if rules allow
- Never same-day triple post

## Realistic outcomes
- Warm 50–100: 80–200 HN points common; 800–3k sessions
- Without comment quality, points stall at 30–50

## Kill / pivot rules
- <10 comments at H+2 → pause cross-posts; fix first comment
- Community asks feel promotional → rewrite maker story

## Ethics line
- No upvote coordination; honest comments only
