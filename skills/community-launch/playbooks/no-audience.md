# Playbook: No Audience (Community Runway)

No HN karma, no IH history, no subreddit presence. Community launch *is* how you earn the right to Show HN — not a one-day hack.

- **Days 1–7: Lurk + contribute on HN.** 3 substantive comments/day on posts in your category. No product links. Build account age and taste profile.
- **Days 8–14: IH build log series (2 posts).** Post #1: problem you felt. Post #2: what you built + screenshot. CTA = profile link only.
- **Days 15–21: Ship demo + QA.** Nothing public until signup works on mobile. Record 60s Loom for future Show HN comment.
- **Day 22: Soft ask in IH — not HN yet.** "Would love feedback from builders" with demo link in comments. Target 30–80 signups.
- **Day 28+: First Show HN** only if demo stable and 10+ organic IH comments received.

Realistic outcome: 21-day runway → first Show HN lands 50–150 points (not front page) but seeds repeat launches.

## Preconditions
- [ ] Marketing profile complete for community-launch
- [ ] Primary metric named before execution
- [ ] Honest aggression dial matches real assets (demo, karma, team hours)

## Aggression dial

| Level | When | Focus |
|-------|------|-------|
| conservative | Thin assets / solo | One channel, learning metrics |
| standard | Warm community | Best-practice cadence from narrative above |
| aggressive | Strong demo + comment history | Execute full tactic stack with kill rules |

## Timeline
- **T-21**: Lurk + comment on HN/IH
- **T-14**: IH build log #1
- **T-7**: Demo QA + title draft
- **H0**: Soft IH milestone (not Show HN yet)
- **D+21**: First Show HN if criteria met

## Tactic stack
1. **`comm_icp_community_map`** — Score 3 communities for ICP fit
2. **`comm_ih_build_log`** — Two posts before any Show HN
3. **`comm_demo_url_live`** — Demo works before public spike
4. **`comm_show_hn_title_draft`** — Draft early; submit later
5. **`comm_hn_comment_sla_60m`** — Practice on others' threads first

## Orchestration
- One primary community for learning; no cross-post spam
- Plan Studio tasks tie to registry tactic ids

## Realistic outcomes
- 21d runway: ready for first Show HN with honest expectations (not front page)
- Skipping runway → thread dies in <2h — predictable failure

## Kill / pivot rules
- No demo by T-7 → postpone all public posts
- Zero IH engagement on build log → rewrite story angle before HN

## Ethics line
- No fake engagement, karma farming, or rule-breaking promo
