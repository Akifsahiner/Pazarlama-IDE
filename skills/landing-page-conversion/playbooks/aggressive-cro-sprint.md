# Playbook: Aggressive CRO Sprint

Competitor teardown + ads↔LP message match. One hypothesis per week with a kill rule.

## Preconditions
- [ ] Live URL with analytics or honest manual counter
- [ ] Primary CTA defined (signup / demo / waitlist)
- [ ] ≥1 competitor URL for teardown

## Aggression dial

| Level | When | Focus |
|-------|------|-------|
| conservative | No traffic | Clarity rewrite only |
| standard | Some traffic | Proof stack + CTA test |
| aggressive | Ads or partner traffic | Weekly hypothesis + message match audit |

## Timeline
- **T-7**: Competitor teardown checklist (5-second test × 3 URLs)
- **T-5**: Rewrite hero for ICP pain → outcome
- **T-3**: Proof adjacent to CTA; kill secondary CTAs
- **H0**: Ship variant A
- **D+7**: Read leading metric; kill or scale
- **D+8**: Variant B if kill

## Tactic stack
1. **`hero_social_proof_stack`** — One primary CTA; proof without scroll on desktop
2. **`lp_competitor_teardown`** — 5-second clarity notes for 3 competitors
3. **`ads_lp_message_match`** — Ad promise = first screen promise (if paid)
4. **`cro_hypothesis_weekly`** — One hypothesis; sample size rule before declare win
5. **`cta_friction_audit`** — Form fields ≤ essential; remove tour links
6. **`mobile_fold_check`** — CTA visible on iPhone SE fold
7. **`honest_baseline`** — Manual conversion OK until GA connected — never invent rows

## Orchestration
- CRO primary; paid creative only after message match pass
- Asset generator for proof screenshots / gallery

## Realistic outcomes
- Clarity-only: +10–30% relative conversion common when hero was vague
- With traffic ≥200 visits/week: detectable A/B in 1–2 weeks
- Aggressive with ads: CPA drop only if message match holds — else honest ceiling

## Kill / pivot rules
- Variant loses after agreed sample → revert same day
- Multiple CTAs fighting → force one primary
- Ads CTR high / LP convert low → fix LP before new creative

## Ethics line
- No fake testimonials, fake logos, or invented GA4 metrics
