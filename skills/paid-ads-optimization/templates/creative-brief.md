# Creative Brief — Paid Ads Template

Message match is non-negotiable: ad headline ≈ LP headline ≈ first screen of creative. Brief designers and copywriters together — siloed assets kill CTR and CVR.

---

## Campaign overview

```
Product:       [Name]
Offer:         [Free trial / Demo / Purchase / Waitlist]
ICP:           [Role, company size, industry]
Channel:       [Meta / Google / LinkedIn / TikTok]
Objective:     [Conversions / Leads / Traffic — pick one]
Target CPA:    $[X]
Live date:     [YYYY-MM-DD]
```

---

## Insight (why this ad exists)

```
[ICP role] cares about [outcome], not [feature category].

Proof point: [metric, customer quote, or demo fact]

Competitive context: [What they're doing now — spreadsheet, incumbent, manual process]
```

**Example:**
```
RevOps leads care about ramp time, not "enablement software."

Proof: Rippling cut onboarding from 21 days to 5.

Competitive context: Notion + Loom + Slack links — no single source of truth.
```

## Creative specs by channel

| Channel | Format | Dimensions | Length | Safe zones |
|---------|--------|------------|--------|------------|
| Meta Feed | Static or video | 1080×1080 or 4:5 (1080×1350) | 15–30s video | Text < 20% of image |
| Meta Stories/Reels | Video | 9:16 (1080×1920) | 15s max for prospecting | Bottom 250px clear |
| Google Display | Static | 1200×628 + 300×250 | — | Logo top-left |
| Google Search | Text only | RSA: 15 headlines, 4 descriptions | — | Pin 2 headlines max |
| LinkedIn | Single image or video | 1200×627 or 1080×1080 | 15–30s | Minimal text on image |
| TikTok | Video | 9:16 | 15–21s | Hook in first 1s |

## Copy blocks (deliver all variants)

### Headlines (5 minimum — for RSA or A/B)

1. [Outcome + timeframe]: "Cut onboarding from 14 days to 2"
2. [Problem callout]: "Your onboarding docs are in 4 tools"
3. [Social proof]: "40 SaaS teams switched last quarter"
4. [Offer]: "Free for teams under 10 — start today"
5. [Question]: "Still onboarding reps with Notion links?"

### Primary text (125 chars visible before truncate on Meta)

```
[Problem in 1 sentence]. [Product] [mechanism] so [ICP] can [outcome]. [Proof]. [CTA].
```

### Description / secondary text

```
[Offer terms]. [One objection handler]. [CTA repeat].
```

### CTA button

```
[Start free trial / Book demo / Get started / Sign up]
— match LP button exactly
```

---

## Reject the creative if it:

- Leads with logo animation (prospecting video)
- Uses headline that LP doesn't repeat — message mismatch drops CVR 20–40%
- Shows UI that doesn't exist in production
- Targets "everyone" in visual casting — show the ICP's environment (office, not beach)

## Ethical notes

- Political/sensitive targeting: follow platform policies; document audience rationale
- Testimonials in ads need permission on file
- Before/after claims in weight loss, finance, etc. trigger special category rules
