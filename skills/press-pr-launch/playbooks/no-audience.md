# Playbook: Press Launch With No Press Relationships

Most founders overestimate cold email to reporters. Without warm intros, press is a **long runway asset** — not a launch-day miracle. Build proof and relationships first; pitch five beat-matched reporters with honest expectations.

## Preconditions
- [ ] Accept honest ceiling: **0–1 pickup** from 5 cold pitches is normal
- [ ] `days_until_launch ≥ 14` (prefer 21+) for non-breaking news
- [ ] Real proof ready: metric, funding, partnership, or 3+ named design partners
- [ ] Willing to invest 10–15h in press kit + personalized hooks (not a wire blast)

## Aggression dial

| Level | When | Focus |
|-------|------|-------|
| conservative | No proof, no demo | Skip press; changelog + community only |
| standard | Proof + 14d runway | This playbook — 5 pitches, modest expectations |
| aggressive | Warm intro + proof + 21d | Upgrade to `aggressive-press-war-room` only with ≥1 relationship |

**Honest ceiling:** Cold list of 5 beat-matched reporters → expect 0–1 stories. Zero pickups is not failure if you learn which hooks resonate; spamming 50 reporters is failure.

## Timeline

| Phase | When | Action |
|-------|------|--------|
| T-21 | −21d | Map 15 reporters by beat; read last 10 stories each; score fit 1–5 |
| T-14 | −14d | Draft press kit PDF + founder quote + screenshot kit |
| T-10 | −10d | Secure customer proof permission (email OK) |
| T-7 | −7d | Shortlist top 5 reporters; draft 5 unique hooks |
| T-5 | −5d | Send personalized pitches with embargo offer |
| T-3 | −3d | One follow-up only to non-responders (+48h rule) |
| T-1 | −1d | Confirm embargo datetime with any interested reporters |
| H0 | Embargo lift | Founder on standby 6h; no social leaks before lift |
| H+6 | +6h | Launch monitoring pass #1 |
| H+48 | +48h | Kill rule: if 0 pickups, pivot to owned media teardown |

## Tactic stack

1. **`pr_honest_coverage_ceiling` (T-21)** — Document expected pickups (0–1 cold) in plan; align team expectations. Metric: ceiling written in launch doc.
2. **`pr_screenshot_asset_kit` (T-14)** — Hero UI, workflow, before/after metric; PNG + optional GIF. Metric: 3 assets in `/press/` folder.
3. **`pr_press_kit_pdf` (T-14)** — ≤8-page PDF: fact sheet, boilerplate, bio, screenshots, quote, embargo footer. Metric: PDF link live.
4. **`pr_founder_quote_approved` (T-14)** — ≤60 words, one insight; founder sign-off in writing. Metric: quote in kit + email sig block.
5. **`pr_customer_proof_permission` (T-10)** — Written OK for any logo/metric/quote in kit. Metric: permission email saved.
6. **`pr_embargo_datetime_lock` (T-7)** — `launch_datetime_utc` in every pitch and kit footer. Metric: datetime in all 5 drafts.
7. **`pr_reporter_shortlist_5` (T-7)** — Five reporters with beat fit ≥4/5; no generic "tech" blasts. Metric: shortlist sheet with last-story notes.
8. **`pr_pitch_personalized_hook` (T-5)** — Each pitch cites reporter's recent work + ICP angle. Metric: 5 sends logged, 0 duplicate body text.
9. **`pr_follow_up_sla_24h` (T-5→D+2)** — Reply to any reporter question ≤24h. Metric: 100% SLA in window.
10. **`pr_launch_monitoring_6h` (H0)** — Track outlets + UTM 6h post-lift. Metric: monitoring sheet updated at H+1, H+3, H+6.
11. **`pr_kill_no_pickup_48h` (H+48)** — Zero stories → stop pitching; publish teardown + email wave. Metric: pivot decision logged.

## Orchestration

- **Primary:** Five personalized reporter emails + press kit PDF (hosted link, not 25MB attachment)
- **Parallel (support):** Email list amplifies *after* embargo lift — never "leak" to list pre-lift
- **Do not:** PR Newswire blast, paid placement, 50-reporter mail merge, LinkedIn pre-announce
- **Community/PH:** Can run same launch week **after** embargo lifts — coordinate timing in plan

## Realistic outcomes

| Profile | Pickups (48h) | Press sessions | Signups | Notes |
|---------|---------------|----------------|---------|-------|
| Cold, strong hook + proof | 0–1 | 0–800 | 0–80 | Reply rate 10–30% is a win |
| Cold, weak hook | 0 | 0 | 0 | Learn; don't blast 50 more |
| 1 warm intro + proof | 1–2 | 500–3k | 40–300 | Intro worth 10× cold email |

Attribution: press traffic often peaks D+1–D+7; measure 14-day window.

## Kill / pivot rules

- T-7: if press kit not ready → postpone pitch 7 days; do not send broken kit
- T-3: if 0 replies from 5 pitches → revise hook or proof; **do not** add 20 more reporters
- H+48: 0 pickups → execute `pr_kill_no_pickup_48h`; invest in teardown + waitlist referral
- Any embargo leak (founder or reporter) → log, adjust future embargo policy; don't blame reporters publicly

## Ethics line

- **Never:** fabricated logos, paid pay-for-play articles, fake exclusives, embargo leaks for X clout
- **Always:** accurate metrics, permissioned customer proof, honest "under embargo until {{datetime}}" footer
- Cold outreach is ethical; volume spam and deception are not
