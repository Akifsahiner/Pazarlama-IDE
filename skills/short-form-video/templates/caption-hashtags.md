# Caption & Hashtag Formulas — Short-Form Video

Caption repeats the hook — many viewers watch muted. Hashtags: 3–5 niche + 1 broad max. Put the CTA in the first line AND last line.

---

## Caption structure (copy this order)

```
Line 1: [Hook restated as standalone sentence — must work without video]

Line 2–3: [One sentence context or proof]

Line 4: [CTA with specific action]

—
[Optional: 3–5 hashtags on TikTok/IG; 2–3 on YouTube Shorts]
```

---

## Template 1 — Direct response (conversion-focused)

```
You're losing [metric] to [mistake]. Here's the fix in 15 sec.

[Product] helps [ICP] [outcome] — [customer name] cut [metric] by [X]%.

Try free → link in bio

—
#[niche1] #[niche2] #[niche3] #saas #founder
```

**Filled example:**
```
You're losing 6 hrs/week to onboarding doc chaos. Here's the fix in 15 sec.

Acme gives RevOps one source of truth — Rippling cut ramp time from 3 weeks to 4 days.

Try free → link in bio

—
#revops #saas #onboarding #startups #b2b
```

## Hashtag selection rules

| Platform | Count | Strategy |
|----------|-------|----------|
| TikTok | 3–5 | 1 broad (#saas), 2–3 niche (#revops), 1 campaign (#productname) |
| Instagram Reels | 5–8 | Mix niche + location if local; hide in comment optional |
| YouTube Shorts | 2–3 in description | Put keywords in title instead — hashtags secondary |
| LinkedIn (if cross-posting) | 3 max | Professional tags only; write caption longer, less hashtag-heavy |

**Niche > volume:** #revops (50k posts) beats #business (50M posts) for B2B discovery.

## CTA variants — rotate weekly

| CTA | Best for |
|-----|----------|
| Link in bio | Default conversion |
| Comment "[keyword]" for the template | Engagement + DM funnel |
| Follow for part 2 | Multi-part series |
| Save this for later | Teach content; boosts algorithm |
| DM me "[word]" | High-touch B2B |

## Cross-platform UTM defaults

| Platform | utm_source | utm_medium |
|----------|------------|------------|
| TikTok | tiktok | organic_video |
| Instagram | instagram | reels |
| YouTube | youtube | shorts |

Example: `https://product.com/?utm_source=tiktok&utm_medium=organic_video&utm_campaign=hook_v1`

---

## Reject the caption if it:

- Has 20+ hashtags (looks spammy; dilutes niche signal)
- Buries the CTA below "see more" on LinkedIn cross-posts
- Makes regulated claims without disclaimer in caption
- Copies competitor captions verbatim — algorithm and audience both punish this

## Ethical notes

- #ad required if video is sponsored — in caption AND on-screen first 3 seconds
- "Results not typical" when showing outlier customer metrics
- Don't gatekeep the teach in caption to force link click unless the video genuinely delivers standalone value
