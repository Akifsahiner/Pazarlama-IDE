# 15-Second Script — Template

One idea. One outcome. Captions burned in. 9:16 native. Cut every 2–3 seconds — static talking head dies at 5s.

**Timing map:** 0–3s hook | 3–10s body | 10–13s proof/demo | 13–15s CTA

---

## Template 1 — Problem → fix (default)

```
[0–3s] HOOK (face + text overlay)
"You're still [painful manual task]?"

[3–7s] AGitate (b-roll or screen)
"Most [ICP plural] waste [metric] on [specific step]."

[7–12s] Demo (screen recording, sped 1.2× if needed)
"[Product]: [action on screen] → [result on screen]"

[12–15s] CTA (face or end card)
"Link in bio — free to try."
```

**Filled example (B2B SaaS):**
```
[0–3s] "Your onboarding docs are in 4 different tools."
[3–7s] "RevOps teams lose 6 hrs/week just finding the latest version."
[7–12s] "Acme: one click → every rep sees the current playbook."
[12–15s] "Link in bio — setup in 10 min."
```

---

## Template 2 — Myth bust

```
[0–3s] "Stop [common advice]."
[3–8s] "It works for [wrong audience], not [your ICP]."
[8–12s] "Instead: [one specific tactic]. We built [product] to do exactly that."
[12–15s] "Follow for more — demo in bio."
```

---

## Template 3 — Pure demo (no face required)

```
[0–2s] Text on screen: "[Outcome] in 15 seconds"
[2–12s] Screen capture only — 4 cuts:
  Cut 1: Open app
  Cut 2: Key action
  Cut 3: Result/metric
  Cut 4: Share or export
[12–15s] End card: [Product name] + "Link in bio"
```

---

## Production checklist

- [ ] Hook text appears in first frame (not fade-in at 1s)
- [ ] Captions: high contrast, bottom third, 2 lines max
- [ ] Audio: voiceover OR trending sound under 20% volume — not both fighting
- [ ] Export 1080×1920, H.264, no letterboxing from horizontal source
- [ ] Safe zone: keep text away from bottom 250px (UI overlay) and top 150px (username)

---

## Batch filming guide (10 scripts in 90 min)

1. Set camera + light once; batch all hooks to camera (30 min)
2. Batch screen recordings separately — same app state, clean demo account (30 min)
3. Assembly in CapCut/Descript: hooks + demos + captions template (30 min)
4. Export all with same caption style for brand consistency

---

## Reject the script if it:

- Explains 2+ features — split into separate videos
- CTA is longer than 3 seconds or asks for follow + like + comment + link
- Demo uses lorem ipsum or fake data that ICP would spot instantly
- Runs 22 seconds "because the demo needed it" — cut the demo
