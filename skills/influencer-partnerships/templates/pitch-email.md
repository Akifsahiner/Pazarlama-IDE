# Creator Pitch Email — Templates

Personalize the first line or don't send. Micro-first: 10k–100k followers with high ICP overlap beats one 500k generic creator.

---

## Template 1 — Micro-creator cold pitch (default)

**Subject:** [Specific video/post title] → partnership idea

```
Hey [first name],

Your [video/post] on [specific topic] — especially [specific moment or take] — is exactly the audience we built [product] for.

Quick context: [Product] helps [ICP] [outcome] without [pain]. [One proof point — beta user, metric, or founder story in one sentence].

We'd love to sponsor a [deliverable: 60-sec integration / dedicated video / newsletter mention]. Structure:

→ [Base: $X / product access / affiliate X% per signup]
→ Creative freedom — we send a one-page brief, you say it your way
→ Unique tracking link + code so you see performance in real time

Not a fit if [honest disqualifier — e.g. you're not using anything in this category]. If it is, happy to send a 90-sec demo Loom before you decide.

Worth a 15-min call this week?

— [your name]
[Title] | [Company]
```

## Template 2 — Warm intro (mutual connection or prior engagement)

**Subject:** [Mutual name] suggested I reach out

```
Hey [first name],

[Mutual name] mentioned you're the right person for [topic] content — and your [recent post] on [topic] confirms it.

We're launching [product] to [ICP]: [one-line outcome]. [Mutual name] thought your audience would genuinely benefit, not just tolerate an ad.

Proposal: [deliverable] with [comp structure]. Full brief is one page; you keep editorial control. We only fact-check claims.

Demo: [Loom link — 90 sec]

Open to a quick call, or I can answer async if you prefer email.

— [your name]
```

## Template 3 — Podcast / newsletter host

**Subject:** Guest + sponsor hybrid for [show name]?

```
Hey [first name],

Long-time listener — [specific episode] on [topic] changed how we think about [related problem].

Two options (happy with either):

A) **Sponsor read:** 60-sec mid-roll for [product]. [Comp: $X CPM / flat $X / rev-share]. Brief + talking points, not a script.

B) **Founder interview:** I can share [contrarian angle / build story / metric] that fits your [audience type] — product mention natural, not forced.

Tracking: unique URL `[show-specific UTM]` + promo code [CODE] for your audience.

Either way, no pressure — reply "pass" and I won't follow up.

— [your name]
```

## Template 4 — Follow-up (one only, 5–7 days later)

**Subject:** Re: [original subject]

```

Short version: [deliverable] for [comp one-liner]. Demo still here: [Loom link]

If it's a no or bad quarter, just ignore — I won't ping again.

— [your name]
```

## Outreach batch checklist (first 10 creators)

- [ ] Unique UTM + promo code created before send
- [ ] Watched/read at least one full piece of their content
- [ ] First line references something specific
- [ ] Comp range fits your CPA model (see `tracking-sheet.md`)
- [ ] Brief link ready if they reply same day

---

## Reject the pitch if it:

- Opens with "I'm a big fan!" without specifics
- Offers "exposure" as primary compensation to creators with 50k+ engaged followers
- Sends attachments over 5MB before they opt in
- Uses a mail-merge obvious template — "[Creator Name]" visible

## Ethical notes

- CAN-SPAM: include physical address in footer for bulk outreach tools
- Don't ask creators to post before product is usable — refunds and chargebacks follow
- Hybrid CPA deals: disclose to creator if payout depends on their audience converting
