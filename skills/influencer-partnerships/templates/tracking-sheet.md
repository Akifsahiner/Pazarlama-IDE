## ICP fit scoring (1–5)

| Score | Criteria |
|-------|----------|
| **5** | Audience IS your ICP; content is directly in problem domain |
| **4** | Adjacent audience; 60%+ overlap with buyer persona |
| **3** | General tech/business; engagement high but ICP fuzzy |
| **2** | Large reach, low relevance — only for awareness plays |
| **1** | Mismatch — do not pursue regardless of follower count |

## Status pipeline

```
Research → Pitched → Negotiating → Brief sent → Draft review → Live → Reporting → Renew/Kill
```

**SLA:** Respond to creator replies within 48h at every stage.

---

## CPA kill / scale rules

Define before first dollar spent:

| Rule | Default threshold | Action |
|------|-------------------|--------|
| **Kill** | CPA > 2× target after $200 spend or 14 days | Pause affiliate; no renewal |
| **Review** | CPA 1.25–2× target | Negotiate creative refresh or shorter integration |
| **Scale** | CPA ≤ target + 20% signups from creator | Renew + offer 3-month ambassador deal |
| **Instant scale** | CPA ≤ 0.5× target | Double comp for second deliverable within 30 days |

**Target CPA:** $[X] (derived from LTV × max CAC ratio, e.g. 30% of 90-day LTV)

## Weekly review checklist

- [ ] Pull clicks + signups from analytics (UTM source/medium = creator slug)
- [ ] Reconcile promo code redemptions vs UTM (discrepancy > 15% → investigate link placement)
- [ ] Flag creators with 500+ clicks and 0 signups — landing page or audience mismatch
- [ ] Update Renew column; send renewal or thank-you-pass email within 7 days of report
- [ ] Log creative learnings: which hook, length, disclosure placement converted best

---

## Reject the partnership if tracker shows:

- CPA rising 3 weeks straight on same creative — audience fatigued, not "bad creator"
- Creator refuses disclosure — non-negotiable
- 80%+ traffic from non-ICP geos when you sell B2B US-only
- Click fraud pattern (spike in 0-second sessions)

## Ethical notes

- Share performance data with creators monthly — transparency builds long-term partnerships
- Don't claw back base fee if CPA misses unless contract specifies performance guarantee
- Promo codes tied to personal data: GDPR/privacy policy must cover referral tracking
