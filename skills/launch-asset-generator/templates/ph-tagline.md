# Product Hunt Tagline — Fill-In Template

**Hard limit: ≤60 characters.** Sentence case. No trailing period. No emojis. Appears next to product name on PH front page — strangers must understand instantly.

---

## Positioning spine (copy from product profile)

```
ICP:              [role + company type]
Outcome:          [measurable result]
Category:         [how buyers search]
Wedge:            [one axis vs incumbent]
```

---

## Pattern picker (choose one)

- [ ] **X for Y** — `[Tool] for [audience/use case]`
- [ ] **Outcome verb** — `[Verb] [object] in [time/unit]`
- [ ] **Replace** — `The [adj] [category] without [incumbent pain]` *(use sparingly)*
- [ ] **Identity** — `For [identity] who [verb] [noun]`
- [ ] **Demoable claim** — `[Number claim] for [audience]`

## Stranger test

Read tagline to someone who doesn't know the product. First question should be about **how it works** or **pricing** — NOT "what does it do?"

```
Tester understood category?     ☐ Yes ☐ No → rewrite
Tester guessed wrong ICP?       ☐ Yes ☐ No → add ICP signal
```

---

## Reject the tagline if:

- Contains: AI-powered, next-gen, revolutionary, seamless, intuitive, platform, solution
- >60 characters
- Could describe 3 competitors equally
- No user or outcome named
- Ends with period or exclamation mark
- Uses jargon a non-customer wouldn't know
