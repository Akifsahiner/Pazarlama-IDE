# Playbook: Aggressive Platform-Native Asset Pack

One export size is not a launch kit. Specs per channel.

## Preconditions
- [ ] Product screenshots available (real UI)
- [ ] Tagline ≤8 words
- [ ] Brand voice line in profile

## Aggression dial

| Level | When | Focus |
|-------|------|-------|
| conservative | Solo | PH gallery + 1 LinkedIn image |
| standard | Multi-channel | Per-platform pack |
| aggressive | Launch week | Full native set + alt hooks |

## Timeline
- **T-10**: Shot list from product flows
- **T-7**: PH gallery 4+ slides
- **T-5**: LinkedIn carousel + OG image
- **T-3**: Short-form cover frames
- **H0**: Final QA against platform checklists

## Tactic stack
1. **`platform_native_asset_pack`** — Dimensions + safe margins per channel
2. Real UI only — no mockup-only galleries for PH
3. Alt hook variants for A/B on social

## Orchestration
- Pair with ph_launch / short-form / LinkedIn playbooks

## Realistic outcomes
- Native packs lift CTR vs cropped generic exports (channel-dependent)

## Kill / pivot rules
- Asset rejected by platform → fix specs same day
- Fake UI in gallery → rebuild before launch

## Ethics line
- No stolen competitor screenshots; no fake logos
